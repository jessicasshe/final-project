Using Prepared Statements:
-> Alternative to single statements

What are they?
Pre-compiled SQL queries with placeholders (?) of the values 

Actual values are added later to enhance security

Steps:

1. Create PreparedStatement stmt objct
2. Call prepareStatement on statement object with quer requesy
3. Set parameter valuees
4. Execute query with ResultSet (for select), or int for insert/update/delete
https://www.geeksforgeeks.org/java/how-to-use-preparedstatement-in-java/
5. Process resources
6. Close result set, statement and connection
